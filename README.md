# Hello

# Requirements
* Fastfetch (https://github.com/fastfetch-cli/fastfetch)
* Git (https://git-scm.com)

# Installation (Unix)
Clone this repository to `.config` folder
```bash
git clone https://github.com/king-of-ducks/fastfetch.git ~/.config/
```
